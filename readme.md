# OpeNAA

A web app to add and keep track of your favoite AA / NA meetings and find recovery related resources. You can create a meeting to share it with other community members and comment on meetings. 

# Tech Stack
- Node
- Express.js
- JavaScript
- Mongoose / MongoDB
- Passport.js
- HTML / CSS
